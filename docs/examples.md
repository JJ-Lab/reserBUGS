# Examples

The repository includes notebooks for complete workflows:

- `notebooks/Example_biotime.ipynb`: BioTIME-style ecological forecasting.
- `notebooks/Example_daily_data_retriever.ipynb`: daily data retrieval workflow.
- `notebooks/Example_NDVI.ipynb`: MODIS NDVI retrieval and integration.

## Sample Data

Sample input data are available in:

```text
data/sample/series_4_RESERBUGS_EXAMPLES.csv
```

## Generated Outputs

Example workflows can create generated files under:

```text
outputs/
```

These files are useful for local exploration, but they are treated as generated
artifacts rather than package source.
