# Citation

If you use reserBUGS, cite the software:

```text
reserBUGS contributors (2026). reserBUGS: Reservoir computing for ecological forecasting.
Zenodo. https://doi.org/XXXX
```

A formal publication describing the method is in preparation.

## Example Datasets

If you use BioTIME data, cite:

```text
Dornelas, M. et al. (2025). BioTIME 2.0: Expanding and Improving a Database of
Biodiversity Time Series. Global Ecology and Biogeography, 34(5), e70003.
https://doi.org/10.1111/geb.70003
```

## Environmental Data

If your work relies on environmental data retrieved through reserBUGS, also cite
the corresponding data providers, such as:

- Copernicus Climate Change Service for ERA5 data.
- NASA LP DAAC for MODIS MOD13A3 data.
